Политика конфиденциальности

Сплит Мастер (далее — «Приложение») уважает вашу конфиденциальность. Настоящая Политика объясняет, какие данные мы собираем, как их используем и как вы можете управлять ими.

1. Какие данные собираются

Приложение собирает следующие данные:

  • Учётная запись пользователя: имя, адрес электронной почты, номер телефона (для приглашений), роль (администратор/сотрудник), идентификатор команды (teamId).
  • Записи клиентов: имя клиента, телефон, дополнительный телефон, адрес, дата и время записи, длительность, заметки, список услуг (наименование, количество, цена), статус (активен/выполнен/отменён).
  • Данные устройства: модель устройства, Android ID (хешированный), идентификатор для push-уведомлений (Player ID от OneSignal).
  • Данные о звонках: при предоставлении разрешения приложение может определять входящие/исходящие номера для отображения информации о клиенте (оверлей).
  • Данные геолокации (только при построении маршрута к клиенту, с вашего согласия).

2. Как используются данные

  • Для аутентификации и синхронизации данных между устройствами (через Firebase Auth и Firestore).
  • Для создания и отображения записей клиентов.
  • Для отправки push-уведомлений о создании, изменении, удалении записи и изменении статуса (через OneSignal).
  • Для отображения информации о клиенте во время звонка (если дано разрешение).
  • Для построения маршрута к клиенту (если дано разрешение на геолокацию).

3. Передача данных третьим лицам

Приложение использует следующие сторонние сервисы, которые могут обрабатывать ваши данные:

  • Firebase (Google) — аутентификация, база данных, облачные функции. Политика конфиденциальности Google
  • OneSignal — push-уведомления. Политика OneSignal
  • Яндекс.Карты (через веб-ссылку) — построение маршрута. Политика Яндекса

Мы не продаём и не передаём ваши данные третьим лицам, кроме случаев, предусмотренных законом.

4. Хранение и безопасность

  • Локальные данные хранятся в зашифрованной базе Room на вашем устройстве.
  • Данные в облаке (Firestore) защищены правилами безопасности Firebase и доступны только участникам вашей команды (пользователям с одинаковым teamId).
  • Передача данных по сети осуществляется через зашифрованные каналы (HTTPS, Firebase SDK).

5. Ваши права

Вы можете:

  • Редактировать или удалять свои записи (в зависимости от прав, заданных администратором).
  • Запросить удаление своей учётной записи через функцию «Удалить аккаунт» в приложении или обратившись к администратору.
  • Отозвать разрешения (звонки, контакты, уведомления, геолокация) в настройках устройства.

6. Дети

Приложение не предназначено для лиц младше 13 лет. Мы не собираем осознанно данные детей.
